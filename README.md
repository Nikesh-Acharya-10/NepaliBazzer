# NepaliBazzer - E-Commerce Platform

## Description

NepaliBazzer is a Java-based e-commerce application that showcases Nepali products including handicrafts and food items. The system demonstrates object-oriented programming concepts with product inheritance, shopping cart management, and local product filtering.

## Features

- **Product Management**: Support for different product types (Handicrafts, Food Products)
- **Handicraft Products**: Track artisan information, craft type, and cultural significance
- **Food Products**: Handle expiry dates, organic certification, and local sourcing
- **Shopping Cart**: Add products to cart and filter local Nepali products
- **VAT Calculation**: Automatic VAT inclusion for food products
- **Local Product Display**: Filter and display products made in Nepal

## Project Structure

```
├── NepaliProduct.java       # Base product class
├── HandicraftProduct.java   # Handicraft product subclass
├── FoodProduct.java         # Food product subclass
├── ShoppingCart.java        # Shopping cart management
└── NepaliBazzer.java        # Main class
```

## Technologies Used

- Java 8+
- Object-Oriented Programming (OOP)
- Inheritance & Polymorphism
- Collections (ArrayList)

## How to Compile and Run

```bash
# Compile
javac NepaliBazzer.java

# Run
java NepaliBazzer
```
